# Tic-Tac-Toe Game 
A classic X-O game developed in C++ where two players can compete against each other or against a bot.

## Features 
- Multiplayer: Play with friends locally.
- Single Player: Play against a bot.
- Score Tracking: Gathering points and saving progress.
- OOP Structure.
  
## Project Structure
The project is divided into two main versions:
1. Procedural Version: A functional approach using basic functions and global logic.
2. OOP Version : Structure using Classes, Encapsulation, and Inheritance.
##Technolgies Used
  - Language: C++
  - Compiler: g++
 Concepts Applied
- Classes and Objects (OOP)
- Functions
- Vectors and Arrays
- File Management (I/O)
  
## How To Run
To compile and run the project, use the following command in your terminal:
- To run the procedural version: 
  g++ main.cpp -o prgame
- To run the OOP version: 
g++ main.cpp src/func.cpp src/include.cpp fileManager/file.cpp -Iinclude -o game
